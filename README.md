Control de notas
=========
Sistema en línea para el control de notas para lael colegio *Visión de Fe*. La aplicación se encuentra disponible [Aquí] [sitio].

Herramientas usadas
--
- Bootstrap 3.2.0
- Jquery
- MySQL
- PHP 5.5
